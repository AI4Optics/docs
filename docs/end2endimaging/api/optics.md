# Optics (DeepLens)

The optical lens models in End2endImaging are provided by
**[DeepLens](../../deeplens/index.md)** — the differentiable optics engine — and
are available as `end2end_imaging.deeplens.*`. Rather than duplicate them here,
they are documented in full in the DeepLens API reference.

[:octicons-arrow-right-24: Go to the DeepLens API Reference](../../deeplens/api/index.md){ .md-button .md-button--primary }

This covers the lens models (`GeoLens`, `HybridLens`, `DiffractiveLens`,
`PSFNetLens`, `DefocusLens`), optical surfaces, light/wave representations, and
image-simulation utilities.
